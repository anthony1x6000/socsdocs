# Contributing 

## Install 

### Node 

- Install [NVM](https://github.com/nvm-sh/nvm#installing-and-updating)
- Install NodeJS LTS: 
```bash
nvm install --lts
```
### Deps 

Install dependencies using [pnpm](https://pnpm.io/):

```bash
pnpm install
```
## Comments
Try to use jsdoc, it has nice vscode integration
